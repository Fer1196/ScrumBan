# Kanban 

Kan se lo puede definir como un tablero visual 

Es un tablero visual que usa tarjetas para ver el estado actual de las tareas. 
Permite la visibilidad de todos los estados de produccion
Evita riesgos y retrasos 

Tarjeta
	Descripción
	Número
	Encargado 
	Por hacer 
	En Progreso 
	Terminado
